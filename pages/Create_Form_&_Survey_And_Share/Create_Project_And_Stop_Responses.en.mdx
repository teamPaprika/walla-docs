## Create a New Project

<Steps>
    ### If you want to create a new form from scratch, click 'Start from Scratch'.
    - Survey templates have not been implemented yet. If you want this feature, please vote for it on the Product Roadmap and Feature Suggestions page.

    ![Untitled](https://paprikastorage.com/images%2FcreateProjectAndStopResponses%2Fen%2F0.png)

    ### Set the 'Project Name' and 'Badge'
    - After creating the survey, you can still change the project name and emoji, but the badge color cannot be changed.

    ![Untitled](https://paprikastorage.com/images%2FcreateProjectAndStopResponses%2Fen%2F1.png)

    ### Click 'Create' to make an new project.

    ![Untitled](https://paprikastorage.com/images%2FcreateProjectAndStopResponses%2Fen%2F2.png)

</Steps>

## Import from Google Forms

<Steps>
    ### You can easily import existing forms you've already created in Google Forms to Walla. Click 'Import from Google Forms'.

    ![Untitled](https://paprikastorage.com/images%2FcreateProjectAndStopResponses%2Fen%2F0.png)

    ### Copy the link from 'Send Form' in your Google Forms.

    ![Untitled](https://paprikastorage.com/images%2FcreateProjectAndStopResponses%2Fen%2F3.png)

    ![Untitled](https://paprikastorage.com/images%2FcreateProjectAndStopResponses%2Fen%2F4.png)

    <Callout type='warning' emoji="‚ö†Ô∏è">**For the 'Import from Google Forms' feature, make sure to disable any sub-features requiring login under Google Form ‚û°Ô∏è Settings ‚û°Ô∏è Response section. Failure to do so may result in unsuccessful form imports.**</Callout>

    ### Simply paste the link into the 'Import from Google Forms' input field in Walla.
    - Set the 'Project Name' and 'Badge'
    - Activating 'Receive Responses via Google Forms' will accumulate responses on both Walla Form and Google Forms.

    ![Untitled](https://paprikastorage.com/images%2FcreateProjectAndStopResponses%2Fen%2F5.png)

    ### Successfully imported Google Forms to Walla!

    ![Untitled](https://paprikastorage.com/images%2FcreateProjectAndStopResponses%2Fen%2F6.png)

</Steps>

<Callout type='info' emoji="üîî">**What changes occur when converting from Google Forms to Walla Forms? - Logic (as of 24.02.14), fields, required input, response conditions, etc. are all imported to Walla. - However, images and videos included in Google Forms fields will not be transferred to Walla. - Google Forms 'File Upload' field will not be transferred to Walla.**</Callout>

## Finish Project: Stop Receiving Responses

<Steps>
    ### To stop receiving responses for the project you want, click on 'Receive Responses', then disable the 'Toggle' in the top right corner.
    - The 'Receive Responses' button can be enabled or disabled in all drawers within the project.
    - Stopping responses based on response count, setting a date and time to stop receiving responses, etc. are on Walla's future development roadmap. If you'd like to see these features immediately, please vote for them here

    ![Untitled](https://paprikastorage.com/images%2FcreateProjectAndStopResponses%2Fen%2F7.png)
</Steps>
