## 히든필드 활용하기

<Callout type='info' emoji="🔔">**히든필드 사용의 기본**</Callout>
1. 프로젝트 편집의 '설정'에서, 히든필드 제목을 설정해주세요.
2. 공유할 링크를 히든필드 형식에 맞게 직접 수정해주세요.
3. 공유한 링크를 통해 응답이 들어오면, 응답시트에서 히든필드가 잘 적용되었는지 확인할 수 있어요.

<Steps>
    ### 히든필드를 넣고자 하는 프로젝트를 열고, '편집하기'의 오른쪽에서 '설정' 탭을 열어주세요.

    ![Untitled](https://paprikastorage.com/images%2FhiddenField%2Fko%2F0.png)

    ### '히든필드 편집하기' 버튼을 누르고, 설정하고자 하는 히든필드의 제목을 필드 안에 작성해주세요.

    <Callout type='info' emoji="🔔">**히든필드 제목 설정에 어려움이 있으신가요?**</Callout>

    - 파악하고 싶은 유입 경로를 큰 카테고리로 묶었을 때, 그 카테고리의 이름이 히든필드의 제목입니다.
        1. 예를 들어, '인스타그램, 이메일, 페이스북, 유튜브'라는 네 개의 서비스에 '사전예약신청서'라는 하나의 설문을 업로드한다고 가정해보겠습니다. 특정 응답이 네 개 중 어떤 서비스에서 유입되었는지를 파악하고 싶다면, 히든필드 제목을 'source', 'service' 혹은 'channel' 등으로 설정해요.
        2. 다른 예시로, 회사 내에서 진행하는 세 개의 캠페인에 하나의 설문을 업로드한다고 가정해보겠습니다. 특정 응답이 세 개의 캠페인 중 어떤 캠페인에서 유입되었는지를 파악하고 싶다면, 히든필드 제목을 'campaign' 등으로 설정해요.
        3. 또 다른 예시로, 다섯 개의 서로 다른 컨텐츠에 하나의 설문을 업로드한다고 가정해보겠습니다. 특정 응답이 다섯 개의 컨텐츠 중 어떤 컨텐츠를 통해 유입되었는지를 파악하고 싶다면, 히든필드 제목을 'contents' 등으로 설정해요.
    - 히든필드 제목에는 특별한 제한이 없으나, 되도록이면 영문 소문자로 입력해주세요. 띄어쓰기가 없는(혹은 '_'로 구분되는) 간단한 형태라면 더욱 좋아요.
    - 또한, '1', 'a' 등의 형태보다는 유입을 묶어 볼 수 있는 큰 카테고리의 이름을 넣어 직관적으로 그 뜻을 파악할 수 있도록 하는 것이 좋아요.

    ![Untitled](https://paprikastorage.com/images%2FhiddenField%2Fko%2F1.png)

    - 회색 필드에 히든필드 제목을 적어주세요. 한 행당 하나의 히든필드가 만들어집니다.
    - '저장하기'를 누르시면 히든필드 제목이 생성됩니다.

    ### 이제, 히든필드 규칙에 맞게 공유할 설문의 링크를 수정해주세요.
    - 히든필드는 설문 응답자가 설문에 어떤 링크로 들어왔는지를 파악하고 응답 시트에 해당 응답의 유입 경로를 반영하는 식으로 작동해요. 따라서, '인스타그램, 이메일, 페이스북, 유튜브'라는 네 가지 서비스에 설문 링크를 공유한다면, 채널별로 서로 다른 네 개의 링크를 만들어야 해요.

    <Callout type='info' emoji="🔔">**히든필드 규칙에 맞게 공유할 설문의 링크를 만드는 법을 알려드릴게요.**</Callout>
    1. 공유할 설문의 링크를 준비합니다. '공유하기' 탭에서 링크를 가져오세요.
        - `https://walla.my/v/QA9S0i0UfZn5rZTfSbd4`
    2. 링크 뒤에 물음표 하나를 붙여주세요.
        - `https://walla.my/v/QA9S0i0UfZn5rZTfSbd4`<strong style={{color: 'red'}}>?</strong>
    3. 물음표 뒤에 앞에서 설정한 히든필드 제목을 적어주세요(제목이 여러 개면 하나만 먼저 적기).
        - `https://walla.my/v/QA9S0i0UfZn5rZTfSbd4?`<strong style={{color: 'red'}}>source</strong>
    4. 히든필드 제목 뒤에, '=' 표시를 붙여주세요.
        - `https://walla.my/v/QA9S0i0UfZn5rZTfSbd4?source`<strong style={{color: 'red'}}>=</strong>
    5. 이 설문 링크를 공유할 구체적인 소스 이름을 적어주세요. (혹은 구체적인 캠페인, 컨텐츠 분류)
        - `https://walla.my/v/QA9S0i0UfZn5rZTfSbd4?source`<strong style={{color: 'red'}}>=instagram</strong>
        - `https://walla.my/v/QA9S0i0UfZn5rZTfSbd4?source`<strong style={{color: 'red'}}>=email</strong>
        - `https://walla.my/v/QA9S0i0UfZn5rZTfSbd4?source`<strong style={{color: 'red'}}>=facebook</strong>
        - `https://walla.my/v/QA9S0i0UfZn5rZTfSbd4?source`<strong style={{color: 'red'}}>=youtube</strong>


    ### 링크를 각 채널에 맞게 공유하고 응답 시트에서 유입 경로를 확인해보세요.
    - 예를 들어, `https://walla.my/v/QA9S0i0UfZn5rZTfSbd4?source=`<strong style={{color: 'red'}}>youtube</strong>라는 링크를 통해 들어와 설문에 응답해보겠습니다.

    ![Untitled](https://paprikastorage.com/images%2FhiddenField%2Fko%2F2.png)

    - 응답시트에 'source'라는 열이 만들어지고, 해당 링크를 통해 들어온 응답은 'youtube'라는 유입 경로를 자동으로 남긴 것을 볼 수 있습니다.

    ![Untitled](https://paprikastorage.com/images%2FhiddenField%2Fko%2F3.png)

    - 그럼 https://walla.my/v/QA9S0i0UfZn5rZTfSbd4?source=<strong>facebook</strong> 라는 링크를 통해 들어가 설문에 다시 한 번 답해보겠습니다.

    ![Untitled](https://paprikastorage.com/images%2FhiddenField%2Fko%2F4.png)

    - 응답 시트에서 응답자가 'facebook' 링크를 통해 접속했다는 것을 확인할 수 있어요.

    ![Untitled](https://paprikastorage.com/images%2FhiddenField%2Fko%2F5.png)

    ### 하나의 설문에 여러 개의 히든필드를 넣을 수도 있습니다.
    - 회사에서 A 캠페인과 B 캠페인을 진행하면서, 이 두 개의 캠페인을 인스타그램과 페이스북에 업로드한다고 가정해볼까요? 우선 캠페인 A, B를 크게 묶는 히든필드 제목은 'campaign'으로, 인스타그램과 페이스북을 크게 묶는 히든필드 제목은 'source'로 설정할 수 있어요.

    <Callout type='info' emoji="🔔">**여러 개의 히든필드를 넣을 때, 설문 링크 작성법**</Callout>

    1. 앞에서 작성한 설문 링크를 가져옵니다.
        - `https://walla.my/v/QA9S0i0UfZn5rZTfSbd4?source=instagram`
    2. 설문 링크 뒤에 '&' 표시를 붙여주세요.
        - `https://walla.my/v/QA9S0i0UfZn5rZTfSbd4?source=instagram`<strong style={{color: 'red'}}>&</strong>
    3. 앞과 동일하게, '히든필드 제목=캠페인 이름'을 붙여주세요.
        - `https://walla.my/v/QA9S0i0UfZn5rZTfSbd4?source=instagram&`<strong style={{color: 'red'}}>campaign=A</strong>

    - 히든필드를 더 달고 싶다면, 규칙은 동일합니다. '히든필드 제목=분류할 이름'을 적고, '&' 표시로 이어주시면 됩니다.

</Steps>

### 상황 배경 :

- '햄버거 선호도 조사'라는 설문을 만들었어요.

1. 인스타그램, 유튜브, 페이스북에서 광고할 거예요. (=3개의 서로 다른 매체)
2. 동영상 광고와 이미지 광고를 시도할 거예요 (=2개의 서로 다른 형식)
3. 광고 문구를 ‘저렴한 가격'과 '육즙이 풍부함’ 두 개로 나눌 거예요. (= 2개의 서로 다른 콘텐츠)

- 응답이 들어왔어요! 담당자는 이런 부분이 궁금해져요:

1. 더 많은 응답자를 모으기 위해, 이 많은 사람들은
    - 인스타그램, 유튜브, 페이스북 중 어떤 매체로 들어왔을까?
    - 동영상과 이미지 중 어떤 형식의 광고를 접했을까?
    - 어떤 광고 문구를 보았을까?

2. 성실히 답변해준 응답자가 있다면, 이 성실한 응답자는
    - 인스타그램, 유튜브, 페이스북 중 어떤 매체로 들어왔을까?
    - 동영상과 이미지 중 어떤 형식의 광고를 접했을까?
    - 어떤 광고 문구를 보았을까?

- 위의 궁금증을 해소하기 위해 두 가지 방법을 사용할 수 있어요.

1. 응답자에게 직접 유입된 매체, 형식, 광고 문구를 작성해달라고 부탁해요.
    - 모든 응답자가 다시 참여해야 해서 해서 불편해요.
    - 모든 응답자가 정확히 답변해주지 않아 파악이 어려워요.

2. 히든필드를 사용해요.
    - 응답자에게도, 회사에게도 훨씬 편리해요!

### 사용 방법 :

<Steps>
    ### '설정' 탭에서 세 개의 히든필드 제목을 만들었습니다.
    - 'source'는 응답시트에서 응답자가 접속한 매체(인스타그램, 유튜브, 페이스북)에 대해 보여줄 거예요.
    - 'form'은 응답시트에서 응답자가 접한 광고 형식(동영상, 이미지)에 대해 보여줄 거예요.
    - 'contents'는 응답시트에서 응답자가 접한 광고 문구(저렴한 가격, 육즙이 풍부함)에 대해 보여줄 거예요.

    ![Untitled](https://paprikastorage.com/images%2FhiddenField%2Fko%2F6.png)

    ### 각 광고 채널에 넣을 링크를 생성합니다.
    - 소스 3개 * 형식 2개 * 컨텐츠 2개 = 12개의 링크를 만들어야 해요.
