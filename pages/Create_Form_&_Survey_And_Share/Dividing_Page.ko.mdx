### 리스트뷰에서 페이지가 구분되어 보여집니다.

<Callout type='warning' emoji="⚠️">**스텝뷰에서는 페이지 나누기가 적용되지 않아요.**</Callout>


    💡 ‘다중필드’와 ‘페이지 나누기’는 무엇이 다른가요?

    | | 다중필드 | 페이지 나누기 |
    | --- | --- | --- |
    | 필드별 로직 | X | O |
    | 필드별 옵션 | X | O |
    - 다중필드는 그 자체가 하나의 필드이기 때문에, 다중필드에 속하는 각각의 필드에 로직을 설정할 수 없어요.
    - 페이지 나누기는 구분선 사이의 필드에 자유롭게 로직을 추가할 수 있어요.

    | | 다중필드 | 페이지 나누기 |
    | --- | --- | --- |
    | 스텝뷰 | O | X |
    - 다중필드는 스텝뷰에서 여러 필드을 한 페이지에 보여줄 수 있어요.
    - 페이지 나누기는 스텝뷰에서는 적용되지 않아, 한 필드씩 볼 수 있어요.

    | | 다중필드 | 페이지 나누기 |
    | --- | --- | --- |
    | 리스트뷰 | X | O |
    - 다중필드는 리스트뷰에서 다른 필드들과 구분되지 않아요.
    - 페이지 나누기는 리스트뷰에서 나누어진 페이지들을 넘기며 볼 수 있어요.

    ⇒ ‘페이지 나누기+리스트뷰’는 ‘다중필드+스텝뷰’와 비슷하지만, 더 나아가 각각의 필드에 옵션을 다르게 설정하거나 로직을 추가할 수 있어요.
