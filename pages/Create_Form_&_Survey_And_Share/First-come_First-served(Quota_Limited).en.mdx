<Callout type='info' emoji="ðŸ””">Quota setting allows you to set a maximum number of selections per option, similar to course registration, and close the option once the maximum number is reached. Quota setting is available starting from the Pro plan.</Callout>

1. In the project editing screen, click 'Set up' in the quota setting section.

![image](https://paprikastorage.com/images/firstComeFirstServed(quotaLimited)/en/1.png)

2. Set the maximum number of responses for each option.

![image](https://paprikastorage.com/images/firstComeFirstServed(quotaLimited)/en/2.png)

3. After saving, click 'Publish'.

![image](https://paprikastorage.com/images/firstComeFirstServed(quotaLimited)/en/3.png)

4. An error message will appear before submission for any option selected by more people than the set limit.

![image](https://paprikastorage.com/images/firstComeFirstServed(quotaLimited)/en/4.png)
